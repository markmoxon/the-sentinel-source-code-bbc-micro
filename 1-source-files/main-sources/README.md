# Annotated source code for the BBC Micro version of The Sentinel

This folder contains the annotated source code for the BBC Micro version of The Sentinel.

* Main source files:

  * [the-sentinel-source.asm](the-sentinel-source.asm) contains the main source for the game

* Other source files:

  * [the-sentinel-disc.asm](the-sentinel-disc.asm) builds the SSD disc image from the assembled binaries and other source files

* Files that are generated during the build process:

  * [the-sentinel-build-options.asm](the-sentinel-build-options.asm) stores the make options in BeebAsm format so they can be included in the assembly process
